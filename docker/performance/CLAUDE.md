# docker/performance/

Performance monitoring stack with Prometheus and Grafana.

## Files

| File | What | When to read |
| ---- | ---- | ------------ |
| `prometheus.yml` | Prometheus scrape configuration | Adding metrics endpoints, modifying scrape intervals |

## Subdirectories

| Directory | What | When to read |
| --------- | ---- | ------------ |
| `grafana/` | Grafana dashboards and provisioning config | Adding monitoring dashboards, configuring data sources |
