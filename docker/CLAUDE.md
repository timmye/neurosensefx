# docker/

Container configurations and orchestration support for deploying NeuroSense FX.

## Subdirectories

| Directory | What | When to read |
| --------- | ---- | ------------ |
| `nginx/` | Nginx reverse proxy configuration | Configuring routing, SSL |
| `postgres/` | PostgreSQL initialization scripts | Setting up database, modifying init schemas |
| `performance/` | Grafana dashboards and Prometheus config for performance monitoring | Setting up monitoring, adding dashboards |
