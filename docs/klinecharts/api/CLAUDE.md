# docs/klinecharts/api/

KLineChart API reference organized by scope — static methods and instance methods.

## Subdirectories

| Directory | What | When to read |
| --------- | ---- | ------------ |
| `chart/` | Static API: registerOverlay, registerXAxis, registerYAxis | Creating custom overlays, building custom axes |
| `instance/` | Instance API: getVisibleRange, scrollToTimestamp, setBarSpace, setFormatter | Manipulating chart viewport, formatting labels |
