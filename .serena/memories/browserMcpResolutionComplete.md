# REMOVED: Browser Tools MCP Server

This MCP server has been removed from the NeuroSense FX project. All references to browser-tools MCP have been removed from the codebase.

## Removal Details
- All browser-tools MCP references removed from run.sh
- All browser-tools MCP references removed from tools.sh

## Reason for Removal
The browser-tools MCP server was no longer needed for the NeuroSense FX project and has been completely removed to simplify the development environment.
