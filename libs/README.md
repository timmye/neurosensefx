# Libraries

This directory contains shared libraries used across the NeuroSense FX project.

> `cTrader-Layer/` is an **internal vendored fork** — checked into this repo and consumed
> via a local `file:` dependency (`@neurosensefx/ctrader-layer`). It is free to modify and
> is **not** installed from npm. Derived from Reiryoku's MIT-licensed cTrader-Layer
> (attribution in its `LICENSE`).

## Structure
- `/cTrader-Layer` - Internal vendored cTrader Open API communication layer (forked, free to modify)
